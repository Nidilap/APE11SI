SEM FLOW

Comando O que faz
git clone Clona o repositório central para a máquina local.
git branch Cria uma nova branch (ramificação).
git checkout Usado para trocar para outra branch ou para um commit anterior.
git status Mostra informações pertinentes sobre a branch atual (o que foi commitado, o que pode ser commitado, pushs, etc).
git add Adiciona um arquivo para ser commitado. Utilizando o parâmetro -a, é possível adicionar todos os arquivos alterados para dar commit.
git commit Cria um commit. É possível inserir uma mensagem usando o parâmetro -m.
git push Salva os commits no repositório central.
git pull Obtem atualizações do repositório central.
git revert Desfaz commits feitos anteriormente.
git merge Faz um merge da branch atual com outra branch (se não for específicado nenhuma branch, faz um merge com a branch master).
git fetch Atualiza toda a branch local.
